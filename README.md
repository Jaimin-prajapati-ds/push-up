# 💪 PushChamp

**AI-Powered Push-Up Counter** — Real-time pose detection using MoveNet to count your push-ups with perfect accuracy.

## Features

- 🎯 **Real-time Rep Counting** — AI detects and counts push-ups automatically
- 📹 **Front Camera Support** — Place phone on floor, face down, and start pushing
- 📊 **Workout History** — Track all your sessions with detailed stats
- 🔥 **Calorie Tracking** — Personalized based on your body weight
- 🏆 **Personal Records** — See your best performances
- 🎵 **Audio Feedback** — Sound effects and voice cues during workout
- 📳 **Haptic Feedback** — Feel every rep with vibration
- 🔒 **No Ads** — Premium experience, zero interruptions
- 📱 **Offline Ready** — Works without internet after first load

## Tech Stack

- React 19 + TypeScript
- TensorFlow.js + MoveNet (pose detection)
- Capacitor (Android native wrapper)
- Tailwind CSS v4
- Vite

## Setup

```bash
npm install
npm run dev
```

## Build Android APK

```bash
npm run build
npx cap sync
npx cap open android
# Build APK/AAB from Android Studio
```
